# ✅ CONCISE CHECKLIST — One-Page Essentials

The TL;DR distilled from [PACKING-LIST.md](./PACKING-LIST.md). If you only read one page before leaving, read this. (Full rationale + sources: [DEEP-RESEARCH.md](./DEEP-RESEARCH.md).)

## 🚲 Bike — Giant Defy (ride-ready before you box it)
- [ ] Tyres **30–32 mm** (widest supple that clears), **tubeless** if possible; sealant topped up
- [ ] Compact crank + wide cassette (~1:1 low gear) confirmed
- [ ] Disc brakes good + **spare disc pads** (correct resin/metal type)
- [ ] Comfortable, broken-in **saddle** (not new); bar tape OK
- [ ] **Mandatory German lights:** white front + red rear, **StVZO, non-blinking**
- [ ] **Bikepacking bags** fitted (saddle/frame/bar — Defy likely has no rack mounts); bell fitted

## 🛠️ Spares & tools (the unbuyable-on-road four ⭐)
- [ ] ⭐ **Bike-specific derailleur hanger** · ⭐ **11-/12-speed quick link** (match your drivetrain) · ⭐ tubeless plugs+sealant (or 2× 700×28–32 tubes) + boot · ⭐ spare disc pads
- [ ] Multi-tool (w/ chain breaker), tyre levers, mini-pump, patch kit, chain lube, spoke key + spokes, zip ties + tape, **carbon-safe torque key**

## 🔒 Security
- [ ] One **high-quality lock** (folding/mini-D) + secure the wheels
- [ ] Bike photos + serial saved offline; bring it indoors every night

## 👕 Clothing (layers + rain + sun, all at once)
- [ ] **2× bib shorts** + **chamois cream** · jerseys/merino tops · base layer
- [ ] **Waterproof jacket** (you *will* use it) · warm layer for 10–12 °C mornings · gloves
- [ ] Helmet · sunglasses · cap/buff · off-bike outfit · swimwear

## 🔌 Electronics
- [ ] Phone + **handlebar mount** + **power bank** + all cables
- [ ] **EU plug adapter (Type F)** · **eSIM installed & tested before flying**
- [ ] Lights charged · **offline maps + GPX downloaded** — the loop is built in [`gpx/`](./gpx) (see [ROUTE-GPX.md](./ROUTE-GPX.md)); load `velo_loop_master.gpx` into Komoot/Polar

## 📄 Documents & money
- [ ] **Passport** valid ≥3 months past mid-July · **ETIAS not needed** (Q4 2026) · expect **EES biometrics** at the border
- [ ] **Travel + cycling insurance** (medical evac + locked-bike theft cover) — number saved offline
- [ ] No-FX-fee card **+ backup** · **~€100–150 cash** (Germany loves cash; shops **closed Sundays**)
- [ ] Offline copies: flights, lodging, passport, **bike receipt/photos**

## 🧴 Health
- [ ] Sunscreen + lip balm · first-aid kit (plasters/blister plasters/gauze) · painkillers/antihistamine/anti-diarrhoeal
- [ ] Prescription meds **in original packaging** · electrolyte tabs · **112** = EU emergency · pharmacy = **Apotheke**

## ✈️ Bike-for-flight
- [ ] Box/case decided + **empty-box plan** (ship/store/buy-at-end) · disc pad spacers · pedals off · bars turned · tyres deflated
- [ ] **Pre-register bike** with airline · reassembly tools in **checked** bag · **no CO₂/fuel** in luggage

---
**The 4 things you cannot replace on the road:** derailleur hanger ⭐ · passport + working card · phone w/ offline maps + power bank · a lock good enough for an insurance claim.
